== KApp Controller Install

Install `kapp-controller`:

Deploy controller:
[source, bash]
----
kapp deploy \
  -f https://raw.githubusercontent.com/jvalkeal/scdf-carvel-poc/main/docs/examples/kap-contoller-install/kc-ghcr.yml
  -y \
  -a \
  kc
----
